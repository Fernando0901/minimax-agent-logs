## Root Cause Analysis

**Síntoma:** El bot muestra `{"tool_call":{"name": "search_client", "arguments": {...}}}` como texto al usuario en lugar de ejecutar la herramienta.

**Causa raíz:** El modelo MiniMax M2.5 no estaba usando correctamente el mecanismo nativo de function calling de la API (`tool_calls` field en la respuesta). En cambio, outputs el tool_call como texto JSON en el `content` del mensaje, siguiendo las instrucciones del system prompt que incluía un ejemplo de formato `{"tool_call": ...}`.

El código en `brain.py:build_system_prompt()` incluía:
```
When you need to use a tool, output a JSON tool_call block in your response.
Example tool_call format: {"tool_call": {"name": "get_price", ...}}
```

Esto provocaba que el modelo outputa tool calls como texto en lugar de usar el mecanismo nativo de la API.

## Changes Made

### 1. `agent/brain.py` — Added text-based tool call detection

Added a **workaround** that detects when MiniMax outputs a tool_call JSON as text in the response content:

```python
# WORKAROUND: If no native tool_calls but response text contains a tool_call JSON,
# parse and execute it (MiniMax-M2.5 sometimes outputs tool calls as text instead of using the tool_calls field)
if not tool_calls and response_text:
    import re
    tc_match = re.search(r'\{[\s]*"tool_call"[\s]*:[\s]*\{', response_text)
    if tc_match:
        # Extract, parse, and execute the tool call from text...
```

The workaround:
1. Detects `{"tool_call": ...}` pattern in response text
2. Extracts the full JSON object using brace matching
3. Parses the tool name and arguments
4. Executes the tool via `execute_tool_call()`
5. Adds the result to messages and continues the loop to get the final response

### 2. `agent/brain.py` — Updated system prompt

Removed the misleading `EXAMPLE_TOOL_CALL` and the instruction to output tool calls as JSON text. Changed from:
```
When you need to use a tool, output a JSON tool_call block in your response.
Example tool_call format: {"tool_call": {"name": "get_price", ...}}
```

To:
```
IMPORTANT: When you need to use a tool, the system will handle tool calls automatically. Just respond naturally and the tool results will be provided to you.
```
